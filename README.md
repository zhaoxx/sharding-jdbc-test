二、查询订单
1）指定购买人id
http://localhost:8080/order/getOrder?buyerId=3
指定数据库节点查询主数据：
SELECT order_id, buyer_id
FROM t_order_0
WHERE buyer_id = 3
UNION ALL
SELECT order_id, buyer_id
FROM t_order_1
WHERE buyer_id = 3
